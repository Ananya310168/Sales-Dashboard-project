# 📊 Madhav E-commerce Sales Dashboard

## 🎯 Objective

Analyze e-commerce sales data to derive insights on revenue, profit, customer behavior, and regional performance using Power BI.

## 🛠 Tools Used

* Power BI
* DAX
* Excel (CSV)

## 📂 Dataset

* Orders Data
* Details Data

## 📈 Key Insights

* Revenue: 438K | Profit: 37K
* Clothing dominates sales volume (~62%)
* COD is the most used payment method (~44%)
* Maharashtra is the top-performing state
* Seasonal fluctuations observed in profit

## ⚙️ Process

1. Data Cleaning & Preparation
2. Data Modeling (Relationships)
3. DAX Calculations
4. Dashboard Creation
5. Insight Generation
